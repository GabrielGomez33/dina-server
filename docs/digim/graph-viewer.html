Load graph data

Paste the JSON from POST /digim/graph (nodes + edges) for Network/Timeline, or from POST /digim/semantic (points) for the Semantic cloud. It auto-detects which, switches view, and ignores surrounding shell text so you can paste a whole terminal block.

Tip: don't jq-filter the semantic response — the cloud needs the full points array. Use | jq . (or no filter), not | jq '{point_count}'.
